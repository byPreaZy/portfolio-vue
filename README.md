 # Portfolio Vue

## Prérequis
- Node.js
- npm (ou yarn)

## Installation
1. Clonez le repository :
   ```bash
   git clone https://github.com/byPreaZy/portfolio-vue.git

## Accedez au repertoire du projet
cd portfolio-vue

## Installer les dépendances
npm install

## Lancer le serveur
npm run serve

Ouvrez votre navigateur et accèdez à 'http://localhost:8080'

## Structue du projet

src/ : Contient les fichiers source du projet.

assets/ : Contient les images et autres ressources statiques.

components/ : Contient les composants Vue.

views/ : Contient les vues principales de l'application.

App.vue : Le composant principal de l'application.

main.js : Le point d'entrée de l'application.

public/ : Contient les fichiers statiques qui seront servis tels quels.

package.json : Contient les dépendances et les scripts du projet.
